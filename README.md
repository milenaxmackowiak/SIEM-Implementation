# SIEM-Implementation
A hands-on cloud security project exploring how to build and deploy a custom ELK Stack SIEM on Microsoft Azure.

# Why this project?
I am building it to bridge the gap between theoretical knowledge within cybersecurity and practical implementations of different commonly utilised solutions.

This project will be used in order to learn the most important concepts of SIEM - components and how they communicate most effectively, logging, normalising the logs, ways of making them most useful and actionable, and many more.

This repository acts as my documentation - in the future it will include all my configuration files, architecture notes, and a step-by-step guide on how I put it all together. 

All feedback and questions are welcome!
