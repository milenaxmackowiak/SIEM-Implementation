# SIEM-Implementation
A hands-on cloud security project exploring how to build and deploy a custom ELK Stack SIEM on Microsoft Azure.

# Why this project?
I built this to bridge the gap between theoretical knowledge within cybersecurity and practical implementations of different commonly utilised solutions . This repository acts as my documentation - in the future it will include all my configuration files, architecture notes, and a step-by-step guide on how I put it all together. Feel free to give feedback or ask questions!
